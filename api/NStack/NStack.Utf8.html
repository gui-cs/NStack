    DecodeLastRune(ustring, Int32)

    DecodeLastRune unpacks the last UTF-8 encoding in the ustring.
    Declaration
    public static (uint Rune, int size) DecodeLastRune(ustring str, int end = -1)
    Parameters
    Type Name Description
    ustring str String to decode rune from; if it is empty, it returns (RuneError, 0). Otherwise, if the encoding is invalid, it returns (RuneError, 1). Both are impossible results for correct, non-empty UTF-8.
    System.Int32 end Scan up to that point, if the value is -1, it sets the value to the lenght of the buffer.
    Returns
    Type Description
    System.ValueTuple<System.UInt32, System.Int32> The last rune and its width in bytes.
    Remarks
    An encoding is invalid if it is incorrect UTF-8, encodes a rune that is out of range, or is not the shortest possible UTF-8 encoding for the value. No other validation is performed.
